# SafeLense Montgomery 🛡️

A neighborhood safety intelligence platform for the City of Montgomery, Alabama.

## What it does
- Resident View: Plain-English safety status per neighborhood
- Official View: Live 911 data analytics dashboard
- Compares media perception vs actual 911 reality
- Live news feed powered by NewsAPI
- Interactive neighborhood heatmap

## Built with
- Google Stitch (UI Design)
- Google AI Studio / Gemini (Development)
- NewsAPI (Live data)
- Leaflet.js (Interactive map)
- Chart.js (Data visualization)

## Challenge Track
Public Safety, Emergency Response & City Analytics

## Data Sources
- City of Montgomery Open Data Portal
- NewsAPI live news scraping
